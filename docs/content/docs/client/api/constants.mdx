## Type Arrays

### `OUTPUT_TYPES`

All output type values defined by the Buttplug protocol:

```ts
import { OUTPUT_TYPES } from "@zendrex/buttplug.js";

// readonly ["Vibrate", "Rotate", "RotateWithDirection", "Oscillate",
//           "Constrict", "Spray", "Temperature", "Led",
//           "Position", "HwPositionWithDuration"]

if (OUTPUT_TYPES.includes(someType)) {
  // valid output type
}
```

### `INPUT_TYPES`

All input (sensor) type values defined by the Buttplug protocol:

```ts
import { INPUT_TYPES } from "@zendrex/buttplug.js";

// readonly ["Battery", "RSSI", "Pressure", "Button", "Position"]
```

## Pattern Constants

### `PRESET_NAMES`

All available preset pattern names:

```ts
import { PRESET_NAMES } from "@zendrex/buttplug.js";

// readonly ["pulse", "wave", "ramp_up", "ramp_down", "heartbeat", "surge", "stroke"]
```

### `PRESETS`

Full preset definitions including keyframe tracks, compatible output types, and default loop behavior:

```ts
import { PRESETS } from "@zendrex/buttplug.js";

const pulse = PRESETS.pulse;
// { description, outputTypes, tracks, loop }
```

### `getPresetInfo()`

```ts
getPresetInfo(): PresetInfo[]
```

Returns metadata for all presets as an array of `PresetInfo` objects:

```ts
import { getPresetInfo } from "@zendrex/buttplug.js";

const presets = getPresetInfo();
// [{ name, description, compatibleOutputTypes, defaultLoop }, ...]
```

## Easing Constants

### `EASING_VALUES`

All supported easing curve names:

```ts
import { EASING_VALUES } from "@zendrex/buttplug.js";

// readonly ["linear", "easeIn", "easeOut", "easeInOut", "step"]
```

### `EASING_FUNCTIONS`

Map of easing names to their implementation functions:

```ts
import { EASING_FUNCTIONS } from "@zendrex/buttplug.js";

const ease = EASING_FUNCTIONS.easeInOut;
ease(0.5); // => 0.5 (midpoint of cubic ease-in-out)
```
