---
title: Introduction
description: Modern TypeScript client for the Buttplug protocol v4
---

**buttplug.js** is a type-safe TypeScript client for the [Buttplug Intimacy Protocol](https://buttplug.io) v4. It connects to a Buttplug server over WebSocket, discovers devices, and controls them through a high-level API.

## Features

<Cards>
  <Card title="Protocol v4" description="10 output types and 5 sensor input types: vibration, rotation, position, oscillation, constriction, and more." />
  <Card title="Type-safe API" description="Full TypeScript types with Zod schema validation for autocomplete and compile-time safety." />
  <Card title="Pattern Engine" description="Keyframe-based pattern playback with built-in presets, custom tracks, easing curves, and loop control." />
  <Card title="Auto-reconnect" description="Configurable reconnection with exponential backoff, attempt limits, and lifecycle events." />
  <Card title="Ping Keep-alive" description="Automatic ping management with configurable intervals to maintain server connections." />
  <Card title="Runtime Agnostic" description="Works in any JavaScript runtime with WebSocket support." />
</Cards>
